### ``PUDIDI GANG``
## `Clone this project`

```php
> git clone https://github.com/PUDlDl/GANG
```

## `Install the dependencies:`
Before running the below command, make sure you're in the project directory that
you've just cloned!!

```php
> cd GANG
> bash install.sh
```

### `Usage..`
```php
> npm start
```
